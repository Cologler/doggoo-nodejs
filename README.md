## How To Use

open cmd.exe, try input:

``` cmd
node main.js https://www.lightnovel.cn/thread-899097-1-1.html
```

if you try to output txt file, try:

``` cmd
node main.js https://www.lightnovel.cn/thread-899097-1-1.html --gen txt
```

### options

#### gen

* txt
* epub (default)
* markdown
